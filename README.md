# HobbyHub

HobbyHub is a **full-stack social networking platform** where users can discover people with similar hobbies, connect with them, and chat in real time.

The platform supports authentication, messaging, media sharing, and background workflows to create a smooth and scalable user experience.

---

# Live Demo

https://hobby-hub-iota.vercel.app/

---

# Features

- User authentication and authorization
- Hobby based user connections
- Real-time chat system
- Media sharing (images and videos)
- Chat notifications
- Recent chats system
- Responsive UI for all devices
- Background workflows with Inngest
- Email services using Nodemailer
- Media optimization using ImageKit

# Installation Guide

## Clone the repository

```
git clone https://github.com/Ayman-Kz25/HobbyHub.git
cd hobbyhub
```

---

# Frontend Setup

```
cd client
npm install
npm run dev
```

---

# Backend Setup

```
cd server
npm install
npm run server
```

# External Services Used

* Clerk – Authentication and user management

* MongoDB – Database

* ImageKit – Media optimization and storage

* Inngest – Background jobs and workflows

* Nodemailer – Email services
